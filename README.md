## 🎮 How to Play

1. Match the **shape** or **number** of the current card
2. Play special cards strategically to disrupt Zeus
3. First to empty their hand wins the round
4. Your **Gaming IQ** is calculated based on:
   - Play accuracy (40%)
   - Win rate (35%)
   - Move efficiency (25%)

### Special Cards

| Card | Effect |
|------|--------|
| **1 (Hold On)** | Safe play, no penalty |
| **2 (Pick Two)** | Opponent draws 2 cards |
| **5 (Pick Three)** | Opponent draws 3 cards |
| **8 (Suspension)** | Opponent loses a turn |
| **14 (General Market)** | Both players draw 1 card |
| **WHOT 20** | Change the shape! |
